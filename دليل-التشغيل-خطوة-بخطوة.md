# دليل تشغيل الموقع ولوحة التحكم — خطوة بخطوة

هاد الدليل لسليم. مكتوب لشخص **مو مبرمج**. اتبع الخطوات بالترتيب.
الهدف: يصير موقعك على الإنترنت (saleemrishah.com)، ومعك لوحة تحكم تعدّل فيها أعمالك بنفسك.

ما رح تحتاج تكتب أي كود. كل شي ضغطات زراير.

---

## نظرة سريعة: شو هي الأجزاء؟

- **GitHub** = "بيت" ملفات موقعك على الإنترنت (مجاني).
- **Netlify** = الخدمة اللي بتاخد ملفاتك من GitHub وبتعرضها كموقع حقيقي (مجاني). كمان بتشغّل لوحة التحكم.
- **لوحة التحكم (Decap)** = صفحة بتفوت عليها (saleemrishah.com/admin)، بتسجّل دخول، وبتعدّل أعمالك بخانات وسحب صور.

الترتيب: GitHub أولاً، بعدها Netlify، بعدها تفعيل اللوحة.

> نصيحة: ما تعمل هالخطوات لحالك إذا حسّيت بأي لخبطة. ارجعلي وبنعملها سوا — بقدر أشوف شاشتك وأدلّك ضغطة ضغطة.

---

## الخطوة 1: حساب GitHub

1. افتح **github.com** واضغط **Sign up**.
2. اكتب إيميلك، اختار باسوورد واسم مستخدم.
   - مهم: الباسوورد بتكتبه إنت بنفسك. أنا ما بكتب باسووردات.
3. أكّد الإيميل (رح يجيك رابط تأكيد).
4. خلص. صار عندك حساب GitHub.

## الخطوة 2: ارفع ملفات الموقع على GitHub

1. بعد تسجيل الدخول، اضغط زر **+** فوق يمين → **New repository**.
2. بخانة الاسم اكتب: `saleemrishah-website`
3. خلّيه **Public**، واضغط **Create repository**.
4. بصفحة المستودع الجديد، اضغط الرابط **uploading an existing file**.
5. افتح مجلد `saleemrishah` على جهازك، اسحب **كل الملفات اللي جوّاه** (مو المجلد نفسه، بل محتواه) للصفحة.
6. انتظر لحد ما ترفع كلها، بعدين اضغط **Commit changes**.

> ملاحظة: الملفات لازم تكون بجذر المستودع. يعني لما تفوت على المستودع لازم تشوف index.html و work.html و مجلد assets مباشرة.

## الخطوة 3: حساب Netlify + ربط الموقع

1. افتح **netlify.com** واضغط **Sign up**.
2. اختار **Sign up with GitHub** (زر واحد، ما بدّك باسوورد جديد).
3. وافق على الربط بين Netlify و GitHub.
4. من لوحة Netlify اضغط **Add new site** → **Import an existing project** → **GitHub**.
5. اختار المستودع `saleemrishah-website`.
6. Netlify رح يقرأ إعداداتك لحالها (في ملف netlify.toml جاهز). اضغط **Deploy**.
7. انتظر دقيقة، رح يطلعلك رابط مؤقت مثل `random-name-123.netlify.app`. افتحه — هاد موقعك صار على الإنترنت!

## الخطوة 4: فعّل لوحة التحكم (Identity + Git Gateway)

هي الخطوة اللي بتخليك تعدّل المحتوى. اتبعها بدقة:

1. بلوحة Netlify، افتح موقعك، روح لتبويب **Identity** → اضغط **Enable Identity**.
2. تحت **Registration**، اختار **Invite only** (حتى ما حدا يسجّل غيرك).
3. انزل لقسم **Services** → **Git Gateway** → اضغط **Enable Git Gateway**.
4. ارجع لتبويب **Identity** → اضغط **Invite users** → اكتب إيميلك → **Send**.
5. رح يجيك إيميل دعوة. افتحه، اضغط الرابط، وحط باسوورد للوحة (إنت بتختاره).
6. خلص! هلأ افتح **رابط-موقعك/admin** (مثلاً `your-site.netlify.app/admin`)، سجّل دخول، ورح تشوف أقسامك الأربعة: AI، Motion، Brand، Football.

## الخطوة 5: اربط الدومين saleemrishah.com

1. بلوحة Netlify → **Domain settings** → **Add custom domain** → اكتب `saleemrishah.com`.
2. Netlify رح يعطيك إعدادات DNS. روح لمكان شراء الدومين (مزوّد الدومين تبعك) وحدّث الإعدادات حسب تعليمات Netlify.
3. انتظر شوي (ساعات أحياناً) لحد ما يشتغل الدومين.

> هي الخطوة فيها تفاصيل حسب مزوّد الدومين تبعك. لما توصلها، ارجعلي وبدلّك حسب مزوّدك بالضبط.

## كيف بتضيف عمل جديد (بعد ما تشتغل اللوحة)

1. افتح `saleemrishah.com/admin` وسجّل دخول.
2. اختار القسم (مثلاً AI & Generative) من اليسار.
3. اضغط **New AI-Projekt**.
4. عبّي الخانات:
   - **Titel** بالألماني والإنكليزي.
   - **Kurzbeschreibung** (وصف قصير) بالألماني والإنكليزي.
   - **Coverbild**: ارفع صورة الغلاف بالسحب.
   - **Tools/Tags**: أضف أدوات (Premiere، Midjourney...).
   - **Video-Link** (اختياري): الصق رابط يوتيوب بصيغة embed.
   - **Weitere Bilder** (اختياري): ارفع صور إضافية تبيّن بالنافذة.
5. اضغط **Publish** فوق.
6. انتظر دقيقة (Netlify بيعيد البناء لحالو)، وحدّث صفحة Work — العمل الجديد بيبيّن ككرت.

**الحذف**: افتح العمل من اللوحة واضغط **Delete**.
**الترتيب**: خانة "Reihenfolge" (رقم) بتحدد الترتيب، الأصغر بيطلع أول.

تقدر تضيف عدد أعمال **غير محدود** بأي قسم. كلها بتنعرض كروت متل بعض.

## ملاحظات مهمة

- روابط يوتيوب لازم تكون بصيغة **embed**: `https://www.youtube.com/embed/VIDEO_ID` (مو رابط المشاهدة العادي). إذا احترت بالصيغة، احكيني وبجهّزلك ياها.
- الأعمال الحالية كلها **مؤقتة** (placeholder) — بدّلها بأعمالك الحقيقية من اللوحة.
- صفحة Contact فيها حساباتك الشخصية بس (جوليا انشالت منها). رابط جوليا بقي داخل كرتها بصفحة Work.
