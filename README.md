# موقع Saleem Rishah — دليل الرفع والتخصيص

## محتويات المجلد
- `index.html` — الصفحة الرئيسية
- `work.html` — الأعمال (مع قسم جوليا والـ Pipeline)
- `about.html` — عني + الخبرات + موديولات velpTEC
- `contact.html` — التواصل + نموذج رسائل
- `danke.html` — صفحة شكر بعد إرسال النموذج
- `impressum.html` و `datenschutz.html` — الصفحات القانونية الألمانية
- `assets/` — الستايل، الجافاسكريبت، الصور

## الرفع على Netlify (الأسهل)
1. افتح https://app.netlify.com وسجل دخول (مجاني).
2. اسحب مجلد `saleemrishah` كامل وافلته على الصفحة (Sites > Drag and drop).
3. الموقع بيشتغل فوراً على رابط مؤقت.
4. لربط دومينك: Site settings > Domain management > Add custom domain > اكتب saleemrishah.com واتبع تعليمات الـ DNS عند مزود الدومين (عادة تغيير A record و CNAME). شهادة SSL بتنفعل تلقائياً.
5. نموذج التواصل بيشتغل تلقائياً على Netlify (Netlify Forms). الرسائل بتوصلك بلوحة التحكم، وفيك تفعل إشعار إيميل من: Site settings > Forms > Form notifications.
   ملاحظة: إذا رفعت الموقع على استضافة غير Netlify، النموذج ما رح يرسل. الحل: احذف الفورم من contact.html أو استخدم خدمة مثل Formspree.

## صور لازم تستبدلها (الحالية Placeholders بنفس الباليت)
- `assets/julia.jpg` — حط أفضل صورة لجوليا (عمودية 4:5، مثلاً 1600x2000).
- `assets/work-ai.jpg` و `assets/work-motion.jpg` و `assets/work-brand.jpg` — صور مشاريعك (4:3).
- `assets/og-cover.jpg` — صورة المعاينة عند مشاركة الرابط (1200x630).
- `assets/portrait.jpg` — صورتك الحالية موجودة، بدلها متى ما بدك.
- الشعار: الملف يلي بعتته كان مربع أسود فارغ. الموقع حالياً مستخدم Wordmark نصي. إذا بدك شعارك: صدّره PNG شفاف بألوان فاتحة وحطه بدل النص بالهيدر.

## تعديل النصوص واللغات
كل نص موجود مرتين:
- النصوص القصيرة: العنصر فيه `data-en="النص الانكليزي"` والنص الألماني داخل العنصر نفسه.
- الفقرات الطويلة: نسختين جنب بعض بكلاسات `only-de` و `only-en`.
عدّل النسختين دايماً. اللغة الافتراضية ألماني، والزر أعلى اليمين بيبدل ويتذكر الاختيار.

## قبل النشر (مهم)
- بصفحة `datenschutz.html`: سجّل اسم مزود الاستضافة الفعلي بالقسم 3 (مكتوب لك Placeholder أصفر بآخر الصفحة كتذكير، احذف التنبيه بعد التعديل).
- راجع بيانات الـ Impressum (العنوان والهاتف منقولين من سيرتك الذاتية).
